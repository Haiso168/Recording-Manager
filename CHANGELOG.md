# 更新日志

本文档记录了通话录音整理工具的所有重要更新。

## [v1.0.0] - 2024-11-XX

### ✨ 新增功能
- 🎯 批量导入录音文件功能
- 🤖 智能分类系统（重要/不重要/待确认）
- 🎵 内置音频播放器（支持快进快退）
- 📱 通讯录导入（支持VCF格式）
- 📊 数据导出功能（JSON格式）
- 🗂️ 文件管理（安全删除功能）
- 🔍 搜索功能（号码和联系人搜索）
- 📋 批量操作支持

### 🎵 支持的音频格式
- MP3 (.mp3)
- M4A (.m4a)
- AMR (.amr)
- WAV (.wav)

### 🛠️ 技术特性
- 使用 PyQt5 构建现代化GUI界面
- 优化的音频时长读取算法
- 多线程处理大量文件
- 响应式表格界面
- 实时搜索和高亮显示

### 📦 分发
- 单文件可执行程序（无需安装Python）
- Windows 10/11 兼容
- 绿色软件，无需安装

## 版本命名规则

我们使用 [语义化版本](https://semver.org/) 控制版本号：

- **MAJOR.MINOR.PATCH** (主版本.次版本.补丁版本)
- **MAJOR**: 不兼容的API变更
- **MINOR**: 向后兼容的新功能
- **PATCH**: 向后兼容的bug修复
